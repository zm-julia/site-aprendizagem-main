Sistema de Arquivos

O sistema de arquivos é a estrutura usada pelo sistema operacional para organizar, armazenar e localizar arquivos e pastas dentro de um dispositivo, como HD, SSD ou pendrive. Ele define como os dados são gravados, como são nomeados e como podem ser acessados pelo usuário.











Modo Texto (CMD)

No modo texto, como no Prompt de Comando (CMD), o sistema de arquivos é acessado por meio de comandos digitados. O usuário navega entre pastas, cria arquivos, move itens e executa operações usando linhas de comando, como cd para mudar de diretório, dir para listar arquivos e mkdir para criar pastas. Tudo é feito de forma textual, sem elementos gráficos.











Modo gráfico(explorer)

No modo gráfico, como no Explorador de Arquivos do Windows, o sistema de arquivos é apresentado por meio de janelas, ícones e menus, permitindo que o usuário navegue e manipule arquivos de forma visual. Aqui é possível abrir pastas com cliques, arrastar arquivos para mover, criar pastas pelo menu e visualizar conteúdos em diferentes formatos, como lista ou miniaturas.
