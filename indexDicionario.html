Dicionário de dados

Um dicionário de dados é um documento que descreve e organiza todas as informações sobre os dados de um sistema. Nele estão registrados os nomes das tabelas, campos, tipos de dados, tamanho, regras, relações e significados. Ele serve como um guia que mostra o que cada dado representa e como deve ser usado, garantindo padrão, clareza e organização no banco de dados.




Tipos de dados de um banco de dados





INT

É um tipo de dado usado para armazenar números inteiros, ou seja, valores sem casas decimais. Ele é ideal para quantidades, contagens, IDs e números que não precisam de frações. É rápido, ocupa pouco espaço e é muito usado em chaves primárias e campos numéricos em geral.











CHAR / VARCHAR

São tipos usados para armazenar palavras, nomes ou pequenos trechos de texto. CHAR guarda um tamanho fixo (sempre ocupa o mesmo espaço), enquanto VARCHAR guarda um tamanho variável, ocupando apenas o espaço necessário. Por isso, VARCHAR é mais usado na maioria das tabelas.











FLOAT / DOUBLE

O Microsoft SQL Server é um banco de dados relacional desenvolvido pela Microsoft, muito usado em empresas. Ele oferece alta segurança, desempenho estável e integração fácil com outras ferramentas da Microsoft, como Azure, Excel e .NET. É ideal para sistemas corporativos, análises de dados e aplicações que exigem confiabilidade. Também possui ferramentas gráficas que facilitam a administração e o gerenciamento do banco de dados.











TEXT

TEXT é um tipo de dado usado para armazenar textos longos, como descrições, comentários, mensagens ou qualquer conteúdo que ultrapasse o limite dos campos curtos (como VARCHAR). Ele permite guardar grandes quantidades de caracteres, sendo ideal quando o tamanho do texto não é fixo ou pode ser muito extenso.











BOOLEAN

O tipo BOOLEAN representa valores lógicos usados para indicar duas opções possíveis: TRUE (verdadeiro) ou FALSE (falso). É usado para armazenar informações como “ativo ou inativo”, “ligado ou desligado”, “sim ou não”. É um tipo simples e muito útil em condições e verificações dentro do banco de dados.











ATE, TIME e DATETIME

DATE, TIME e DATETIME são tipos de dados usados para armazenar informações de datas e horários.

DATE guarda apenas a data (ano, mês e dia).

TIME armazena somente o horário (hora, minutos e segundos).

DATETIME combina os dois, registrando data e hora juntos em um único campo.
