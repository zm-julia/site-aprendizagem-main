Booleano

O tipo booleano representa valores de lógica:

verdadeiro

falso

Ele é usado quando o algoritmo precisa tomar decisões, como verificar se algo é maior, menor, igual ou se uma condição foi atendida.

Exemplos:

Se idade ≥ 18 → verdadeiro

Se senha está correta → verdadeiro ou falso

É essencial em estruturas como se / senao, enquanto e repita.











Caractere (caracter)

O tipo caractere armazena somente um caractere, sempre entre aspas simples.

Exemplos:

'A'

'9'

'-'

'@'

É útil quando o algoritmo precisa trabalhar com letras isoladas, dígitos individuais ou símbolos.











Cadeia (string)

A cadeia é uma sequência de vários caracteres, formando textos completos. É escrita entre aspas duplas.

Exemplos:

"Olá, mundo"

"Maria"

"Senha incorreta"

Usada para guardar nomes, mensagens, frases e qualquer tipo de texto.











Inteiro

O tipo inteiro armazena números sem parte decimal. São valores usados para contagens, idades, quantidades, etc.

Exemplos:

0

25

-12

1500

Inteiros são usados quando não há necessidade de números fracionados.











Real

O tipo real armazena números com casas decimais. Serve para representar valores fracionados ou cálculos mais precisos.

Exemplos:

3.14

7.5

-2.99

É útil em situações que envolvem pesos, notas, medidas, distâncias, etc.











Vetor

O vetor é uma estrutura que guarda vários valores do mesmo tipo, organizados em posições numeradas (índices).

"Pense nele como uma “caixa com várias gavetas”.

Exemplo:

vetor[5] de inteiro

Significa que esse vetor guarda 5 números inteiros, do índice 1 ao 5.

Uso comum:

Lista de notas

Lista de nomes

Lista de idades

Armazenar vários dados de um mesmo tipo

Vetores facilitam repetições e manipulação de grandes quantidades de dados.
