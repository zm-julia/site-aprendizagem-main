Web

A Web (World Wide Web) é um sistema global de informações acessadas pela internet, composto por páginas interligadas por hiperlinks. Essas páginas podem conter textos, imagens, vídeos, áudios e outros tipos de mídia, permitindo que os usuários naveguem, pesquisem e interajam com conteúdos de qualquer lugar do mundo. O acesso à Web é feito por meio de navegadores, como Chrome, Firefox ou Edge, que solicitam informações a servidores que hospedam essas páginas.

A Web é uma das principais formas de uso da internet e possibilita a criação de sites, redes sociais, plataformas de vídeo, lojas virtuais, blogs e diversos serviços online. Ela funciona como uma camada de informação sobre a infraestrutura física da internet, que inclui cabos, roteadores e servidores.

Além disso, a Web utiliza protocolos como HTTP/HTTPS para transmitir dados de forma segura e padronizada, e tecnologias como HTML, CSS e JavaScript para estruturar, estilizar e tornar as páginas interativas. Essa combinação de elementos faz da Web uma ferramenta essencial para comunicação, aprendizado, comércio e entretenimento na era digital.
