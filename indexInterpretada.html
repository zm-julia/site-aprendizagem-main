Linguagem de programação interpretada

É uma linguagem cujo código é executado linha por linha por um interpretador, sem precisar gerar um arquivo separado. Isso facilita testar e corrigir o programa rapidamente. Exemplos de linguagens interpretadas são Python, JavaScript e PHP. A desvantagem é que geralmente são mais lentas que as linguagens compiladas.











Linguagem de programação compilada

É uma linguagem cujo código é transformado inteiro em um arquivo executável por um compilador antes de ser executado. Isso faz com que o programa rode mais rápido e de forma mais eficiente. Exemplos de linguagens compiladas são C e C++. A desvantagem é que, a cada alteração no código, é necessário recompilar para gerar um novo executável.
