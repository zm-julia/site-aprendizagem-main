Variáveis

Variáveis são espaços na memória de um programa usados para guardar informações que podem mudar ao longo da execução. Elas funcionam como “caixinhas” com nome, onde você pode colocar um valor, retirar e colocar outro quando quiser.

Exemplo:

A pontuação de um jogo muda conforme o jogador joga → é uma variável.

A temperatura lida por um sensor → varia o tempo todo, então é variável.

Variáveis são essenciais para programas que precisam lidar com informações dinâmicas, como dados de usuários, cálculos, entradas do teclado, valores recebidos de sensores, etc.











Constantes

Constantes também são espaços usados para guardar valores, mas com uma diferença importante: seu valor é definido uma vez e nunca mais muda durante o programa. Ou seja, é como uma “caixa lacrada”: você coloca o valor no início e ele permanece igual até o fim.

Exemplo:

O número de dias da semana (7) não muda.

O valor de PI (3,14…) é sempre o mesmo.

Uma taxa fixa usada em um cálculo permanece constante.

Constantes são usadas para garantir que certos valores importantes do programa não sejam alterados acidentalmente.
