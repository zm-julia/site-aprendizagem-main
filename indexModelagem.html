Modelagem de banco de dados

Modelagem de banco de dados é o processo de organizar e estruturar as informações que serão guardadas em um sistema. Ela define quais dados existirão, como serão armazenados, como se relacionam e quais regras precisam seguir, garantindo um banco eficiente, organizado e fácil de manter.













DER – Diagrama Entidade-Relacionamento

O DER é um desenho que representa visualmente o banco de dados. Ele mostra:

Entidades (como Cliente, Produto, Pedido)

Atributos (nome, preço, data…)

Relacionamentos (um cliente faz vários pedidos)

É usado para planejar antes de criar as tabelas.











MER – Modelo Entidade-Relacionamento

O MER é o conjunto de conceitos teóricos que explica como entidades, atributos e relacionamentos funcionam.

Ou seja: MER = teoria, conceitos

DER = desenho, representação visual

Eles são usados juntos para criar um banco de dados bem estruturado.
